<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| UI → Drift | START ROUND tap initiates a write to local SQLite via `RoundDao.insertRound` |
| Process lifecycle → persisted state | OS kill mid-insert (extremely narrow window — Drift insert is single transaction) |
| Path param → route handler | `:roundId` deserialized via `int.parse(state.pathParameters['roundId']!)` — non-numeric values crash the route builder |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
