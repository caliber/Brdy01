- GIR putts-per-GIR: sum putts only for holes where `h.greenInRegulation == true`, divide by girCount.
- FIR eligible holes: exclude holes where par == 3 from the denominator count.
- FIR%: if eligible holes == 0, return 0.0.
- GIR%: `girCount / holesWithOutcome * 100` (0.0 if holesWithOutcome == 0).

**File 2: lib/features/round_review/providers/whs_differential_provider.dart**

Define plain Dart class `WhsDifferential` with fields:
```
String displayValue     // formatted string: "12.3", "N/A", or differential number
bool isIndicative       // true when fewer than 18 holes have outcomes
bool isUnavailable      // true when courseRating or slope is null
String? indicativeLabel // "Indicative — fewer than 18 holes" or null
```

Add a `const` constructor.

Define `whsDifferentialProvider` as `@riverpod Future<WhsDifferential> whsDifferential(Ref ref, int roundId)`.

Algorithm per CONTEXT.md and CLAUDE.md:

1. Fetch course: `final course = await ref.watch(courseForRoundProvider(roundId).future)`.
2. Fetch holes: `final holes = await ref.watch(holeListProvider(roundId).future)`.
3. Fetch round from DB for handicapIndex: `final db = ref.watch(appDatabaseProvider); final round = await db.roundDao.getRoundById(roundId)`.
4. If `course?.courseRating == null || course?.slope == null`: return `WhsDifferential(displayValue: 'N/A', isUnavailable: true, isIndicative: false, indicativeLabel: null)`.
5. Count scored holes (`h.outcome != null`). If fewer than 18: set `isIndicative = true`, `indicativeLabel = 'Indicative — fewer than 18 holes'`.
6. Compute `playingHandicap = (handicapIndex × (slope / 113) + (courseRating − par)).round()` per CLAUDE.md. Use `course!.slope`, `course!.courseRating!`, and `course!.par` (already null-checked in step 4).
7. Compute AGS (Adjusted Gross Score): for each of the 18 holes (use course.holes for SI and par), find the matching Drift Hole by holeNumber. If hole has no outcome, use `par + 2 + handicap_strokes_on_hole` as score (ESC net double bogey cap). If hole has outcome: raw stroke count = `par + offset`; cap at `par + 2 + handicap_strokes_on_hole`. `handicap_strokes_on_hole` = 1 if `playingHandicap >= strokeIndex`, else 0 (where strokeIndex comes from `course.holes[i].strokeIndex ?? 18`).
8. Pickup holes: CONTEXT.md says count as `par + 2 + handicap_strokes_on_hole` — this IS the ESC cap, so pickup already hits the cap. Apply the same cap to all outcomes for correctness.
9. `differential = (AGS − courseRating) × 113 ÷ slope`.
10. Format with `NumberFormat('0.0')` from `intl` package to one decimal place.
11. Return `WhsDifferential(displayValue: formatted, isIndicative: isIndicative, isUnavailable: false, indicativeLabel: indicativeLabel)`.
