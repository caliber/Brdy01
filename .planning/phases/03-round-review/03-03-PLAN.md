**File 1: lib/services/share/share_service.dart**

Create `ShareService` as a plain Dart class (not a Riverpod provider — screen calls it directly).

```
import 'dart:ui' as ui;
import 'package:flutter/material.dart';
import 'package:flutter/rendering.dart';
import 'package:share_plus/share_plus.dart';
```

Method: `Future<void> shareScorecard(BuildContext context, GlobalKey key) async`

Implementation:
1. `final boundary = key.currentContext?.findRenderObject() as RenderRepaintBoundary?;`
2. If `boundary == null`, return early without throwing.
3. `final image = await boundary.toImage(pixelRatio: 3.0);`
4. `final byteData = await image.toByteData(format: ui.ImageByteFormat.png);`
5. If `byteData == null`, return early.
6. `final bytes = byteData.buffer.asUint8List();`
7. `final xFile = XFile.fromData(bytes, mimeType: 'image/png', name: 'brdy_scorecard.png');`
8. `await Share.shareXFiles([xFile], subject: 'BRDY.01 Round Scorecard');`

No error rethrow — share failures are non-fatal (user may cancel the share sheet).

| Boundary | Description |
|----------|-------------|
| Screen → Native Share Sheet | Bytes from RepaintBoundary screenshot are passed to OS share API — OS handles file receiver trust |
| SpeechToText → ShotCaptureScreen | External SDK result (bool available) controls UI state only; no score recording depends on voice state |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-06 | Information Disclosure | ShareService — scorecard bytes shared via native OS share sheet | accept | User explicitly initiates share; OS handles recipient selection; no server involved |
| T-03-07 | Denial of Service | RepaintBoundary.toImage() with large scorecard — potential OOM on low-RAM devices | mitigate | pixelRatio capped at 3.0; scorecard is a fixed 18-row Table, bounded size; catch async error in shareScorecard and return early |
| T-03-08 | Elevation of Privilege | PopScope(canPop: false) — prevents back navigation | accept | Intentional design per D-06; user has explicit "Start New Round" exit path |
| T-03-09 | Tampering | speech_to_text.initialize() network call on initState | mitigate | SHOT-13 only reads bool result; SpeechToText object is local to _initVoice, not stored as field; no sensitive data flows through voice init |
| T-03-SC | Tampering | npm/pip/cargo installs | accept | No new package installs; share_plus and screenshot already audited in pubspec.yaml |
</threat_model>
