## Decisions Made
- SQLite does not support ALTER COLUMN, so ON DELETE CASCADE required full table recreation (create-copy-drop-rename). Column names in the raw SQL were verified against drift_schema_v1.json to match Drift's snake_case mapping exactly (T-06-01 mitigation).
- shots table recreation placed after holes in onUpgrade because shots FK references holes — order matters.
- PRAGMA foreign_keys stays in beforeOpen, not onUpgrade. Drift automatically disables FK enforcement during the migration window.
- Used explicit `OrderingTerm(expression: ..., mode: OrderingMode.desc)` constructor per RESEARCH.md Assumption A4, not the `.desc` shorthand.
