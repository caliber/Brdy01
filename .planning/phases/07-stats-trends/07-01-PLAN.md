- ../../../features/round_review/providers/whs_differential_provider.dart

Annotate with @riverpod (lowercase — auto-dispose). Function signature:
  `Future<List<FlSpot>> trendChart(Ref ref) async`

Implementation (follow research Pattern 1):
1. `final rounds = await ref.watch(completedRoundsProvider.future);`
2. Take the last 20 by slicing with `.take(20).toList()` — watchCompletedRounds returns desc order, so take(20) gives the 20 most recent.
3. Reverse to oldest-first: `.reversed.toList()` — so x:0 = oldest round, x:19 = most recent.
4. Loop with index i; for each round call `await ref.watch(whsDifferentialProvider(round.id).future)`.
5. If `diff.isUnavailable` is true, skip (do not increment a separate counter — the x-axis gap is acceptable for this v1).
6. Otherwise: `final v = double.tryParse(diff.displayValue); if (v != null) spots.add(FlSpot(i.toDouble(), v));`
7. Return spots.
