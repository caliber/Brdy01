   ```
   };
   ```

4. Locate the shot-counter `Text('$currentHoleShots', ...)` widget (the 100sp numeral inside the `Stack` at the centre of the chevron row). Wrap it with the flutter_animate tint chain using `key: ValueKey(flashOutcome)`. The animation structure:
   - `.animate(key: ValueKey(flashOutcome))`
   - `.tint(color: _outcomeFlashColor(flashOutcome), end: flashOutcome != null ? 0.4 : 0.0, duration: 300.ms, curve: Curves.easeOut)`
